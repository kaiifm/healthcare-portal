# 🏥 Healthcare Portal

A modern healthcare web application built using React and Vite.
This project provides users with information about hospitals, doctors, treatments, and appointment booking.

## 🚀 Features

* 🏥 Hospital listing
* 👨‍⚕️ Doctors information
* 💊 Treatments details
* 💰 Cost estimation
* 📅 Appointment booking
* 📞 Contact page

## 🛠️ Tech Stack

* React
* Vite
* React Router

## 📂 Project Structure

src/
├── assets/Images
├── components/
├── pages/


## ▶️ Run Locally

```bash
npm install
npm run dev
```
